## 🔧 环境准备

### 系统要求

- **操作系统**: Windows 10+, macOS 10.12+, Linux (Ubuntu 16.04+)
- **Python**: 3.7 - 3.10 (推荐3.8或3.9)
- **内存**: 至少8GB RAM
- **GPU**: NVIDIA GPU (可选，用于加速训练)
- **存储**: 至少5GB可用空间

### 检查Python环境

```bash
python --version
```

如果没有安装Python，请从[Python官网](https://www.python.org/downloads/)下载安装。

## 📦 安装指南

### 方案一：使用pip安装（推荐）

1. **克隆或下载项目**
   ```bash
   # 如果从git仓库克隆
   git clone [repository-url]
   cd TensorFlow2Demo

   # 或直接下载并解压项目文件
   ```

2. **创建虚拟环境（推荐）**
   ```bash
   # 创建虚拟环境
   python -m venv tf2_env

   # 激活虚拟环境
   # Windows:
   tf2_env\Scripts\activate
   # macOS/Linux:
   source tf2_env/bin/activate
   ```

3. **安装依赖**
   ```bash
   pip install -r requirements.txt
   ```

4. **验证安装**
   ```bash
   python -c "import tensorflow as tf; print('TensorFlow版本:', tf.__version__)"
   ```

### 方案二：使用Anaconda安装

1. **安装Anaconda**
   - 从[Anaconda官网](https://www.anaconda.com/products/distribution)下载安装

2. **创建conda环境**
   ```bash
   conda create -n tf2_demo python=3.8
   conda activate tf2_demo
   ```

3. **安装TensorFlow和其他依赖**
   ```bash
   # 安装CPU版本
   conda install tensorflow numpy matplotlib pandas scikit-learn pillow

   # 如果有NVIDIA GPU，安装GPU版本（需要CUDA和cuDNN）
   conda install tensorflow-gpu numpy matplotlib pandas scikit-learn pillow
   ```

4. **安装Jupyter Notebook**
   ```bash
   conda install jupyter
   ```

## ⚠️ 常见问题

### 1. TensorFlow安装问题

**问题：** `ImportError: No module named 'tensorflow'`

**解决方案：**
```bash
# 重新安装TensorFlow
pip install --upgrade --force-reinstall tensorflow

# 或者尝试安装特定版本
pip install tensorflow==2.13.0
```

### 2. GPU相关问题

**问题：** GPU不被识别或CUDA错误

**解决方案：**
```bash
# 检查GPU是否可用
python -c "import tensorflow as tf; print(tf.config.list_physical_devices('GPU'))"

# 如果输出为空，说明GPU不可用，请安装CPU版本：
pip uninstall tensorflow-gpu
pip install tensorflow
```

### 3. 数据下载问题

**问题：** 数据集下载失败

**解决方案：**
- 检查网络连接
- 尝试使用代理或VPN
- 手动下载数据集到正确目录

### 4. 内存不足问题

**问题：** 运行大模型时内存不足

**解决方案：**
- 减小批次大小
- 使用更小的模型
- 增加虚拟内存或系统内存

### 5. 版本兼容性问题

**问题：** 各种库版本冲突

**解决方案：**
```bash
# 创建全新的虚拟环境
python -m venv new_tf_env
source new_tf_env/bin/activate  # Linux/Mac
# 或
new_tf_env\Scripts\activate  # Windows

pip install -r requirements.txt
```

## 🎓 学习建议

### 初学者路径

1. **基础入门** (1-2周)
   - 运行 `01_Basics/` 中的所有示例
   - 重点理解张量操作和即时执行
   - 练习基本的变量和梯度计算

2. **神经网络基础** (2-3周)
   - 运行 `02_Neural_Networks/` 中的示例
   - 学习Keras API的三种使用方式
   - 理解激活函数、损失函数和优化器

3. **数据处理** (1-2周)
   - 运行 `03_Data_Processing/` 中的示例
   - 掌握tf.data API的使用
   - 学习数据增强和预处理技术

### 进阶学习者路径

1. **模型训练技巧** (2-3周)
   - 运行 `04_Model_Training/` 中的示例
   - 学习自定义训练循环
   - 掌握各种回调函数的使用

2. **实际应用** (3-4周)
   - 运行 `05_Practical_Cases/` 中的案例
   - 尝试修改参数和架构
   - 应用到自己的数据集

### 深入学习建议

1. **自定义实现**
   - 尝试实现自己的层和模型
   - 编写自定义损失函数和指标
   - 创建自定义的训练循环

2. **性能优化**
   - 学习使用TensorBoard
   - 优化数据管道性能
   - 实验不同的正则化技术

3. **项目实践**
   - 选择一个实际问题
   - 应用学到的技术
   - 记录和分享学习过程
