# TensorFlow 2.0 学习项目结构说明

本文档详细说明了项目的正确目录结构，避免混淆和重复。

## 📁 正确的项目结构

```
TensorFlow2Demo/                    # 项目根目录
│
├── 01_Basics/                     # 基础语法知识模块
│   ├── tensorflow_basics.py         # 张量、变量、自动微分等基础概念
│   └── eager_execution_demo.py     # 即时执行演示
│
├── 02_Neural_Networks/            # 神经网络基础模块
│   ├── neural_networks_basics.py   # 层、激活函数、损失函数等
│   └── custom_layers_models.py     # 自定义层和模型
│
├── 03_Data_Processing/             # 数据处理模块
│   ├── data_processing_basics.py   # 数据预处理和基础操作
│   └── tfdataset_examples.py      # tf.data API实用示例
│
├── 04_Model_Training/             # 模型训练与评估模块
│   └── model_training_basics.py    # 训练、评估、保存和加载
│
├── 05_Practical_Cases/            # 实用案例模块
│   ├── image_classification.py     # 图像分类案例
│   └── nlp_text_classification.py # 文本分类案例
│
├── 06_Utils/                     # 工具函数模块
│   ├── learning_tools.py          # 可视化、评估、监控工具
│   └── __init__.py              # 模块初始化
│
├── data/                         # 数据存储目录
│   └── (存放数据集和样本文件)
│
├── models/                       # 模型存储目录
│   └── (存放训练好的模型)
│
├── notebooks/                    # Jupyter笔记本目录
│   └── (存放交互式笔记本)
│
├── requirements.txt              # 项目依赖列表
├── README.md                    # 项目总体说明
├── 运行指南.md                 # 详细运行指南
├── demo.py                     # 快速演示脚本
├── 项目结构说明.md              # 本文件
└── .idea/                      # PyCharm配置文件（自动生成）
```

## 🗂️ 目录说明

### 01_Basics/ - 基础语法知识
- **tensorflow_basics.py**: 介绍TensorFlow 2.0的核心概念
  - 张量(Tensors)基础操作
  - 变量(Variables)和自动微分
  - 即时执行(Eager Execution)
  - 函数装饰器(tf.function)

- **eager_execution_demo.py**: 即时执行的详细演示
  - 与图模式的性能比较
  - 调试优势
  - 与NumPy的集成

### 02_Neural_Networks/ - 神经网络基础
- **neural_networks_basics.py**: 神经网络构建基础
  - Keras API三种使用方式
  - 常用网络层详解
  - 激活函数、损失函数、优化器
  - 简单分类模型训练示例

- **custom_layers_models.py**: 自定义层和模型
  - 自定义层实现
  - 自定义模型子类化
  - 自定义训练循环

### 03_Data_Processing/ - 数据处理
- **data_processing_basics.py**: 数据处理基础
  - tf.data API基础操作
  - 数据预处理技术
  - 图像和文本数据处理
  - CSV数据处理

- **tfdataset_examples.py**: 高级数据处理示例
  - 文本数据处理流水线
  - 图像数据处理流水线
  - 时间序列数据处理
  - 多模态数据处理

### 04_Model_Training/ - 模型训练与评估
- **model_training_basics.py**: 完整的模型训练流程
  - 模型编译技巧
  - 训练回调函数
  - 自定义训练循环
  - 模型评估和指标
  - 模型保存和加载

### 05_Practical_Cases/ - 实用案例
- **image_classification.py**: 图像分类实战
  - 基础CNN模型
  - 迁移学习(MobileNetV2)
  - ResNet实现
  - 数据增强技术

- **nlp_text_classification.py**: 文本分类实战
  - 词嵌入技术
  - RNN/LSTM模型
  - 1D CNN文本分类
  - 预训练嵌入使用

### 06_Utils/ - 工具函数
- **learning_tools.py**: 辅助学习工具
  - 可视化工具(训练曲线、混淆矩阵等)
  - 模型评估工具
  - 性能监控工具
  - 数据处理工具

## 🚨 重要提示

### 避免重复结构
请确保项目结构如上述所示，**不要**创建嵌套的`TensorFlow2Demo/TensorFlow2Demo`目录结构。每个模块应该直接位于项目根目录下。

### 正确的工作目录
运行代码时，请确保当前工作目录是`TensorFlow2Demo/`，而不是任何子目录。

```bash
# 正确的工作目录
cd E:\IDEWorkplaces\PyCharm\TensorFlow2Demo

# 运行基础模块示例
python 01_Basics/tensorflow_basics.py

# 运行神经网络示例
python 02_Neural_Networks/neural_networks_basics.py
```

### 导入路径
如果在项目中导入工具函数，确保正确设置Python路径：

```python
import sys
import os
sys.path.append(os.path.dirname(os.path.abspath(__file__)))

# 然后可以导入工具
from 06_Utils.learning_tools import plot_training_history
```
